RAFI LABS — INSTALLATION NOTES

RC AI POWERED CLOUD SECURITY PLATFORM
Windows v1.3.1
- Extract RC-AI-Cloud-Security-Platform-Windows-v1.3.1.zip completely.
- Double-click Install_RC_CloudSec.bat. Do not run it inside the ZIP preview.
- Use 64-bit Python 3.11 or 3.12 and keep internet connected for first installation.
- Open RC AI Cloud Security from the new desktop shortcut.
- Connect only a dedicated AWS identity. Never use the AWS root access key.
- AKIA keys use Access Key + Secret Key with Session Token empty.
- ASIA temporary credentials require Access Key + Secret Key + Session Token from the same session.
- Use RCCloudSecReadOnlyPolicy.json for scanning.
- RCCloudSecRemediationPolicy.json is optional and permits selected S3 and Security Group fixes; attach it only after approval.
- Run fixes only on AWS accounts you own or are explicitly authorized to change.

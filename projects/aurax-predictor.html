AuRa X Prediction Tool

A Windows-to-Android XAUUSD decision workflow built around consistency, safety gates and validation.

MT5PythonWindows BackendAndroid DisplayRisk Gates
Prediction is treated as a decision-support output. Weak or conflicting conditions should produce WAIT instead of a forced trade.

Overview

The AuRa X Prediction Tool is an MT5-connected workflow for processing live XAUUSD data and presenting a simplified BUY, SELL or WAIT decision with supporting evidence and risk levels.

Architecture

MT5 live price, candles, spread and tick activity
↓
Python feature calculations and multi-timeframe context
↓
Grouped decision engine
↓
Spread, freshness, volatility and conflict gates
↓
Windows result API
↓
Read-only Android dashboard

Decision logic

  • BUY: higher-timeframe context, structure, location and live flow are sufficiently aligned.
  • SELL: the independent evidence groups align in the bearish direction.
  • WAIT: stale data, excessive spread, weak evidence, overextension, news risk or timeframe conflict.

Phone and desktop consistency

All calculations are performed once on the Windows backend. The phone displays that same server result instead of calculating its own percentage, reducing cross-device disagreement.

How it is measured

Signals are logged with entry, stop and targets. The system records whether entry was triggered, which level was reached first and how performance changes across trend, range and high-volatility regimes.

Limitations

More features do not automatically create accuracy. Correlated indicators can duplicate evidence, market regimes change and broker-specific feeds differ. Ongoing paper validation is required before real-money use.
